# Propósito

RƎVELA es un laboratorio creativo para descubrir, construir y transmitir la identidad de una persona, marca, producto, servicio o proyecto.

Su finalidad no es generar contenido.

Su finalidad es descubrir una identidad y convertirla en un Sistema Creativo capaz de producir comunicación coherente en cualquier formato.

---

# Principios

La creatividad dirige.

La inteligencia artificial acompaña.

La tecnología nunca sustituye el criterio creativo.

La IA actúa como especialista.

El Director Creativo toma las decisiones.

# Regla principal

Cada etapa tiene una única responsabilidad.

Nunca mezclar varias etapas dentro del mismo módulo.

---

# El Sistema Creativo

El Sistema Creativo es la única fuente de verdad del proyecto.

Todo contenido futuro nace desde él.

Nunca desde un prompt aislado.

Nunca desde un modelo concreto.

---

# Modelos de IA

Los modelos son herramientas.

Nunca forman parte del Core.

Pueden cambiar.

El método permanece.

---

# Especialistas

Los especialistas analizan.

No deciden.

Nunca sustituyen al Director Creativo.

---

# Director Creativo

Coordina.

Relaciona.

Decide.

Construye criterio.

Nunca depende de un proveedor de IA.

---

# Producción

Todo contenido debe derivar del Sistema Creativo.

No se crea contenido desde cero.

Se desarrolla a partir de una identidad ya revelada.

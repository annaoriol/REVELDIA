# ESTILO

El estilo es la forma visible de una identidad.

Convierte la voz y la mirada en una experiencia reconocible.

No es una moda.

Es un sistema coherente de decisiones.

Puede evolucionar sin perder su esencia.

---

# El estilo responde

- ¿Cómo se ve?
- ¿Cómo se siente?
- ¿Qué ritmo tiene?
- ¿Qué atmósfera crea?
- ¿Qué emociones transmite?
- ¿Qué lo hace reconocible?

Construir un sistema visual, sonoro y narrativo coherente que haga reconocible cualquier creación del proyecto.

# HOME

> La Home es la puerta de entrada a REVELA.

Su objetivo es que el usuario sepa inmediatamente dónde está, qué puede hacer y cómo continuar.

No muestra todas las funciones.

Muestra el siguiente paso.

---

# Objetivos

La Home debe transmitir:

- claridad
- inspiración
- calma
- progreso

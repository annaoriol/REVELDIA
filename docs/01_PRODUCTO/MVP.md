# MVP · REVELA

## Objetivo

Construir una primera versión funcional que permita crear un proyecto de principio a fin.

---

## Flujo

1. Crear un espacio de trabajo.
2. Crear un proyecto.
3. Definir o importar recursos.
4. Entrar en el Laboratorio Creativo.
5. Crear una pieza.
6. Guardar automáticamente todo en la Biblioteca.

---

## Funciones

- Espacios de trabajo
- Proyectos
- Recursos
- Laboratorio Creativo
- Equipo Creativo
- Biblioteca

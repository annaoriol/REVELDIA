# EXPERIENCIA DE USUARIO

## El usuario siempre mantiene el control.

REVELA acompaña el proceso creativo.

El usuario puede:

- crear libremente;
- incorporar especialistas cuando lo necesite;
- reutilizar recursos existentes;
- compartir recursos entre proyectos;
- construir una biblioteca viva.

La experiencia debe sentirse como un estudio creativo profesional donde todas las herramientas y especialistas colaboran alrededor del proyecto.

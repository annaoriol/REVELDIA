# ARQUITECTURA · REVELA

> REVELA es un laboratorio creativo inteligente que acompaña a personas, marcas y proyectos desde el descubrimiento de su identidad hasta la creación, producción y publicación de contenidos.

---

# Estructura

REVELA se organiza en cinco grandes áreas.

1. Espacios de trabajo
2. Proyectos
3. Recursos
4. Laboratorio Creativo
5. Biblioteca

---

# Espacios de trabajo

Cada usuario puede disponer de uno o varios espacios de trabajo.

Ejemplos:

- Marca personal
- Empresa
- Agencia
- Cliente
- Productora
- Editorial

Cada espacio contiene proyectos independientes.

---

# Proyectos

Cada proyecto dispone de su propia identidad.

Ejemplos:

- Marca
- Producto
- Servicio
- Empresa
- Campaña
- Evento
- Libro
- Curso
- Documental
- Podcast
- Canal
- Cliente
- Colección
- Lanzamiento

Cada proyecto puede generar una o muchas piezas de comunicación.

# Laboratorio Creativo

Es el espacio donde ocurre la creación.

El usuario puede crear libremente o incorporar especialistas cuando los necesite.

# Biblioteca

La biblioteca reúne todo el conocimiento generado.

Puede almacenar:

- documentos
- imágenes
- vídeos
- música
- prompts
- referencias
- proyectos
- plantillas

Todo permanece conectado con el proyecto que le dio origen.

---

# Principios

- La creatividad pertenece al usuario.
- La IA amplifica el proceso creativo.
- La identidad evoluciona con cada proyecto.
- Los recursos pueden heredarse y compartirse.
- El conocimiento permanece vivo.

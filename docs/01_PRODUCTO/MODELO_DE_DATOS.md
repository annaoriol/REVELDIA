# MODELO DE DATOS · REVELA

## Espacio de trabajo

Es el contenedor principal.

Puede pertenecer a:

- una persona
- una marca
- una empresa
- una agencia
- una productora
- una editorial

Contiene uno o varios proyectos.

---

## Proyecto

Es la unidad principal de trabajo.

Cada proyecto dispone de:

- Identidad
- Recursos
- Piezas
- Biblioteca
- Historial
- Conversaciones
- Especialistas

---

## Identidad

La identidad representa la esencia del proyecto.

Incluye:

- Voz
- Mirada
- Estilo
- Propósito
- Valores
- Personalidad
- Público
- Posicionamiento

La identidad evoluciona continuamente.

---

## Recursos

Elementos reutilizables.

Ejemplos:

- Storytelling
- Prompts
- Moodboards
- Paletas
- Tipografías
- Música
- Storyboards
- Guiones
- Referencias

Los recursos pueden compartirse entre proyectos.

---

## Pieza

Resultado de un proceso creativo.

Ejemplos:

- Reel
- Carrusel
- Post
- Vídeo
- Podcast
- Landing
- Presentación
- Dossier

Cada pieza pertenece a un proyecto.

---

## Especialista

Perfil creativo.

Puede colaborar en cualquier proyecto.

Trabaja utilizando el contexto completo del proyecto.

---

## Biblioteca

Almacena todo el conocimiento generado.

Todo queda relacionado automáticamente con el proyecto.

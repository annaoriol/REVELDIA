# LABORATORIO CREATIVO

> El Laboratorio Creativo es el espacio donde las ideas se transforman en proyectos, contenidos y experiencias.

No representa una herramienta.

Representa un estudio creativo.

El usuario mantiene siempre el control del proceso.

REVELA acompaña cuando el usuario decide incorporarlo.

---

# Objetivo

Facilitar la creación de cualquier proyecto creativo desde una única experiencia.

Todo el proceso permanece conectado con la identidad del proyecto.

---

# Punto de partida

Cada sesión comienza con una intención.

Por ejemplo:

- Quiero crear una marca.
- Quiero crear un reel.
- Quiero crear un documental.
- Quiero crear un podcast.
- Quiero crear una campaña.
- Quiero crear una presentación.
- Quiero crear una web.
- Quiero crear un curso.

---

# Crear libremente

El usuario puede trabajar sin ayuda.

Escribir.

Organizar.

Importar referencias.

Crear recursos.

Modificar documentos.

Todo permanece conectado con el proyecto.

---

# Incorporar especialistas

Cuando el usuario necesita ayuda puede incorporar uno o varios especialistas.

Ejemplos:

- Estratega
- Storyteller
- Director Creativo
- Director de Arte
- Grafista
- Diseñador Editorial
- Diseñador de Títulos
- Director
- Director de Fotografía
- Editor
- Compositor
- Diseñador Sonoro
- Especialista en Publicación

Cada especialista trabaja utilizando el contexto completo del proyecto.

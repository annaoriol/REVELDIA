# DESCUBRIR LA IDENTIDAD

> La identidad no se inventa. Se descubre.

REVELA acompaña a personas, marcas y proyectos a reconocer aquello que ya existe y convertirlo en una forma de comunicar coherente.

La identidad evoluciona con el tiempo.

Cada conversación, proyecto y decisión aportan nuevas evidencias que enriquecen esa identidad.

---

# Objetivo

Descubrir la esencia de un proyecto para construir una comunicación auténtica y coherente.

El proceso debe sentirse natural.

La identidad aparece mediante la conversación, la observación y la creación.

# Aprendizaje continuo

La identidad nunca se considera terminada.

Cada proyecto confirma, modifica o amplía el conocimiento existente.

REVELA mantiene un historial de evolución para comprender cómo cambia un proyecto con el tiempo.

---

# Reutilización

El usuario puede decidir qué elementos compartir entre proyectos.

Ejemplos:

✓ Voz

✓ Mirada

✓ Estilo

✓ Paleta

✓ Tipografía

✓ Storytelling

✓ Prompts

Cada nuevo proyecto puede comenzar desde cero o heredar únicamente los recursos seleccionados.

# Principios

La identidad se descubre.

La conversación genera conocimiento.

La creación confirma la identidad.

Los recursos permanecen conectados.

Cada proyecto fortalece el siguiente.

La creatividad siempre pertenece a la persona.

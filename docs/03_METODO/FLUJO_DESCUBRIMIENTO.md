# FLUJO DE DESCUBRIMIENTO

> Cada proyecto comienza de una forma diferente.

REVELA adapta el proceso de descubrimiento al momento en el que se encuentra el usuario.

No existe un único camino.

---

# Punto de partida

Al crear un proyecto, REVELA pregunta:

## ¿Cómo quieres empezar?

🌱 Descubrir la identidad

🧪 Crear directamente

📥 Importar información existente

---

# Opción 1 · Descubrir la identidad

REVELA inicia una conversación.

Puede hacer preguntas sobre:

- propósito
- valores
- objetivos
- referentes
- personalidad
- estilo
- público

El proceso evoluciona de forma natural.

---

# Opción 2 · Crear directamente

El usuario comienza a trabajar.

Durante el proceso, REVELA identifica patrones y propone mejoras.

La identidad se construye progresivamente.

---

# Opción 3 · Importar

El usuario puede aportar información existente.

Ejemplos:

- web
- dossier
- PDF
- redes sociales
- vídeos
- podcast
- documentos
- imágenes
- presentaciones

REVELA analiza el contenido y genera una primera propuesta de identidad.

---

# Evolución

En cualquier momento es posible:

- ampliar la identidad
- modificarla
- validar propuestas
- compartir recursos
- reutilizar recursos
- comparar versiones

---

# Resultado

Cada proyecto dispone de un ADN propio.

Ese ADN alimenta:

- Laboratorio Creativo
- Equipo Creativo
- Recursos
- Biblioteca

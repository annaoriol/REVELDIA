# IDENTIDAD

La identidad es el núcleo de cualquier proyecto.

No representa una marca únicamente.

Puede pertenecer a:

- una persona
- una empresa
- un producto
- un servicio
- un libro
- un documental
- un podcast
- una campaña
- una comunidad
- un cliente

La identidad evoluciona.

Nunca se considera terminada.

REVELA ayuda a descubrirla, construirla y fortalecerla a medida que el proyecto crece.

---

## Componentes

- Voz
- Mirada
- Estilo
- Propósito
- Valores
- Personalidad
- Público
- Posicionamiento

---

## Principios

Toda creación nace de una identidad.

La identidad guía todas las decisiones creativas.

Los proyectos pueden compartir total o parcialmente una identidad.

La identidad aprende y evoluciona con cada proyecto.

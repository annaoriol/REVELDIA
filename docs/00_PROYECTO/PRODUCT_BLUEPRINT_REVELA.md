# PRODUCT BLUEPRINT · REVELA

## Propósito

Este documento define la arquitectura funcional de REVELA.

No describe pantallas.

No describe tecnología.

Describe la experiencia que vive una persona y las herramientas que necesita para transformar una idea en una identidad capaz de crear, comunicar y evolucionar con autonomía.

Cada etapa debe producir un resultado tangible.

Cada resultado prepara la siguiente etapa.

La experiencia completa es más importante que cualquiera de sus herramientas por separado.
